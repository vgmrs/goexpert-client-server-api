# goexpert-client-server-api
Exercise Client Server API for postgraduate Go Expert

## Description

This application implements a client server API using Golang.

## How to run

To run client:
```
make start-client
```
or:
```
go run client/client.go
```

To run server:
```
make start-server
```
or:
```
go run server/server.go
```
